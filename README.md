## Instalação

Para o funcionamento do projeto, precisamos ter o node instalado, podemos baixá-lo em [NodeJs](https://nodejs.org/en). <br>
Após o Node instalado, no diretório do projeto rodamos os seguintes comandos:

- npm install

## Para Rodar

- npm run dev

## Tecnologias
